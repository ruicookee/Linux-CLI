# Bandit Level 0 --> 1

## Goal
Find password stored in a file called readme in home directory

## Commands
`ls` to see what files exist\
`cd` changes directory\
`cat` prints what is inside a file\
`file` identifies file type\
`du` disk usage\
`find` searches for files

## Process
- ls to see what files are in current directory (flat view)
- readme is within same directory, so just cat `readme`

## Takeaways
`ls -l` to see detailed list of files\
`ls -a` include hidden files starting with .

`cd foldername` eg. `cd Desktop/borbette`\
`cd ..` go up one level\
`cd ~` go to home directory

`cat readme` prints whatever is in the file readme

`find . -name filename` finds file in current directory\
`find . -type f`\
`find . -size 100c`\
`find / -name readme` finds in root (everything)\
`find ~ -name readme` find in home\
`find /tmp -name readme` finds in folder /tmp
